# Nice Hair Core

Core plugin scaffold for the Nice Hair project.

## Intended responsibilities

- Page seed / bootstrap logic
- Popup CPT and popup rendering layer
- Future custom Gutenberg blocks
- Shop-specific business logic and pricing
- API endpoints for custom UI flows

## Current state

This archive contains only the starting structure and bootstrap files.
Block build tooling is prepared as a separate package so theme assets and block assets stay decoupled.
